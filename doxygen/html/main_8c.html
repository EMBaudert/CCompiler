char* get_file_basename ( const char *  file)

Determines the basename of the file, though without file extension.

Parameters
fileThe file name.
Returns
The basename of the specified file. If the file name does not contain any extension, the input filename is returned. However, the string returned by this function is always a newly allocated memory block (malloc) which needs to be freed by the caller.

It returns NULL if the input parameter is NULL or if memory could not be allocated.
